### 快速安装
1. git clone https://github.com/anyvicp2/AnyVICP2-2-PLUS
2. 你需要 Python Django Nginx
3. 打开 AnyVICP2 下方的 config.json 配置
4. 直接 python3 manage.py runserver 80 即可
5. 需要配置 NGINX 反代理, 不多说
